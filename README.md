# Energy Optimization IoT Project

## Overview
This project collects real-time energy data from an Air Conditioner using PZEM and EC200U and stores it in a MySQL database.

## Tech Stack
- MySQL
- Flask (Python)
- IoT Hardware (PZEM + EC200U)

## Features
- Real-time data ingestion
- API-based storage
- Scalable database design

## API Endpoint
POST /ingest

## How to Run
1. Install dependencies
2. Run app.py
3. Send data using test.py or hardware
